## 설치 및 실행

npm install
npm start

## 환경설정

1. `.env.example`을 `.env`로 복사
2. API Gateway URL 입력

## 기술 스택

- React
- AWS Lambda
- Amazon Bedrock (Claude3 하이쿠 + 소넷)
